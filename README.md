# Git Heat Map

## Usage

```
Usage:
  gitmap [flags]

Flags:
  -h, --help             help for gitmap
      --ignore strings   list of file patterns to ignore
      --json             json output format
      --limit int        limit the number of log entries
      --reverse          reverse the order of the output
      --threshold int    threshold to include entries in the output (default 1)
```
